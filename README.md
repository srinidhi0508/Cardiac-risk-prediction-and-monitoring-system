# 💓 Cardiac Monitoring System (PulseGuard)

An AI-powered cardiac monitoring dashboard that analyzes patient heart rate data and predicts cardiovascular risk.

---

## 🚀 Features

* 📊 Real-time heart rate monitoring
* 📈 Graph-based heart rate analysis
* 🧠 Risk prediction (Low / Medium / High)
* 📂 Patient dashboard (Vitals, Risk, Reports)
* ☁️ MongoDB database integration
* ⚡ FastAPI backend + Next.js frontend

---

## 🛠 Tech Stack

* Frontend: Next.js, TypeScript
* Backend: FastAPI (Python)
* Database: MongoDB Atlas
* Visualization: Charts / Graphs


### Frontend

```bash
cd cardio-ai-frontend
npm install
npm run dev
```
